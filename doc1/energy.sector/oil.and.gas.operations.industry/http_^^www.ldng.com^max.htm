Tips For Better Viewing
During the development of this site we have attempted to provide an optimal viewing experience for as many different computer configurations as possible. There are, however, certain guidelines that you can follow to ensure that the time you spend here at Louis Dreyfus Natural Gas is the best that it can be.
   
  To aid navigation, this site is now "frames enhanced", which means that the viewing area is divided up into separate areas called frames. The Louis Dreyfus Natural Gas site use 3 frames: the top banner and the bottom navigational frames which do not change, regardless of where you go within the site; and the main frame which contains the actual page for viewing.

To take advantage of this enhancement, however, your browser must support "frames". Versions of the most popular browsing software (both of which support frames and tables) can be downloaded by following the links above.
  After you have selected "Frames" or "No Frames," and your choice is being displayed, we suggest that you bookmark (add to favorites) that page so that next time you visit, you can go straight there, without having to go through the front door again.
  Regardless of which version of the site you decide on, you must use a browser capable of viewing tables. Much of the information contained in this site is formatted using tables. If your web browser is not capable of viewing tables, you can download a new version by following the links above.
  Although screen resolutions vary widely, we recommend at least 800x600 at 256 colors, however, 1024x768 w/16 bit color (or higher) is preferable.
  Font settings are "browser independent" which means that you set the font type and size within the browser you are using. If you are using a browser where you can actually set the font size, (such as Netscape Navigator) it is recommended that you use a font size of 12pt or 14pt. If you are using a browser that gives choices of small, medium, large, etc., (such as Microsoft Internet Explorer) we recommend that you select medium.
  If you have any questions, problems, (buttons/links that don't work, etc.) suggestions, etc. you can E-Mail the Webmaster directly at KIMMEDA@LDNGC.COM.
  Enjoy your visit, and come back often!
