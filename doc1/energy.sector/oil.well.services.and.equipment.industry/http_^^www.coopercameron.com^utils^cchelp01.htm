Modem Connection

To improve the quality of our communication, our web site incorporates graphics which may slowdown the download of some pages if you are using a slow modem connection. Please make sure that you are connected to the World Wide Web through a modem that has a minimum access speed of 28.8 Kbps.




Browse and Search for Information

Browse: All of the component sites containing Cooper Cameron divisional information are optimized to take advantage of the viewing capabilities available through version 2.0 of the Netscape Navigator Web Browser and/or version 2.0 of the Microsoft Internet Explorer. In addition to viewing HTML (HyperText Markup Language) documents utilizing features of the latest 3.x specification of HTML, these browsers support the integration of "Helper" applications such as Adobe Acrobat Reader and Quicktime Movie Player.

Search: A sophisticated database has been integrated with the site in order to allow us to provide you with quick and efficient access to up-to-date product information. From within the various levels of the site (e.g., Corporate, Industry Segment, or Division) you can click on the Search button to invoke a query tool which allows them to search by a specific product name or choose from among brands and/or product families.




Viewing PDF Documents

Adobe Acrobat Reader: This software reads electronic documents created from a wide range of authoring tools for sharing across different computer platforms. "Printing" files in the Adobe Portable Document Format (PDF) allows you to view, navigate and print documents using the hardware and software resident on your computer. By linking to Adobe Acrobat, you can download the latest version.




Viewing Video and Animations

To review the video movies and animations being used on this Site, you will need to install on your station the following Netscape and Internet Explorer "Helper" applications or plug-ins. Windows 95 users will be able to play the video files using the .AVI format, directly from the Video for Windows AVI component of their operating system.

 

  • Video for Windows AVI
  • Apple QuickTime
  • Macromedia Shockwave

Printing Information

Printing options are controlled entirely through the built-in capabilities of your browser. Within your browser you can choose Print from the File menu, or click the Print icon on the toolbar to print the current HTML document you are viewing. This will print both the text and graphics contained in the document. If you have the Adobe Acrobat Reader installed with your Netscape browser, you will be able to print the PDF documents accessible throughout the site. Both of these printing options support color output if you have the appropriate driver software and printer hardware. For additional help on printing see the Help section of your browser.
