Rolladen Box Machine

This machine is for manufacturing rolladen and angles by foaming expandable polystyrene (EPS) in the appropriate tools. It consists essentially of the tool holder, the tool (mold), a hydraulic system, a vacuum system, two pressure fillers, an electrical system including PLC, a safety system and the system for the pipework and hoses.

The tool holder consists of a stationary frame and moving side and front walls, as well as a swivel cover. The tool holder holds the tool which has connections for steam, condensates, cooling water and granulated EPS.

The tool is opened and closed hydraulically and enables profiles to be inserted. The vacuum system consists of a condenser with two vacuum pumps and a steam trap connected to it. The condensate container has a ventilation valve, safety and outlet valves, a temperature sensor and three level sensors.

The two pressure fillers each have a safety valve set to 2 b. Thez are each equipped with a level transmitter for granulated EPS.

The entire production process is fully automated. The operator can enter control data on the operator's panel.

Manual operation is available for initial setup and testing.
