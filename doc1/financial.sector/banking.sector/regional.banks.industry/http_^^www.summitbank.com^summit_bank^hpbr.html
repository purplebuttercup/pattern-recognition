Having difficulty viewing www.summitbank.com?

Visitors to the Summit Bank web site who experience difficulties viewing the site are most often using an older web browser that is incompatible with some of the features and/or the level of security required to properly view the site and all its features.

Updating your current browser with the most up-to-date version (release) will generally resolve any access problems you may be experiencing. www.summitbank.com is best viewed with Netscape Navigator 2.0 (or higher), or Microsoft Internet Explorer 3.0 (or higher).

You may view www.summitbank.com with any browser that supports frames and uses Secure Socket Layer (SSL) technology.

The following common browsers support SSL:

  • Netscape Navigator 2.0 (or higher)
  • Microsoft Internet Explorer 3.0 (or higher)
  • America Online AOL 3.0
