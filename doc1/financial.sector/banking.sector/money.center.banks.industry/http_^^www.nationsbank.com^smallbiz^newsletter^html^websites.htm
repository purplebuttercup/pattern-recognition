NetBITS (www.netbits.net) is a free, weekly newsletter that contains useful information and tips for navigating and understanding the Web. Each issue features stories that talk about different aspects of the Internet, including practical advice such as how to set up an ISDN connection in your office. You can read the newsletter on screen,  or you can subscribe and  receive the newsletter via e-mail.
