Continuous Machinery Monitoring Systems Available for Plant-wide 24-hour Monitoring

Zonic's Continuous Machinery Monitoring System allows static and dynamic data from critical machinery to be displayed, recorded, and interrogated simultaneously. Plant operators can see machine train graphics, acknowledge alarms, record critical data, and print reports, simultaneous to the engineering team doing in-depth diagnostics work on the same critical machine. This System increases efficiency, and reduces cost.

Since the System is based on a multi-channel parallel signal processor, it can be expanded to a virtually unlimited channel count. We have supplied integrated Turn-key systems that monitor 500 dynamic and 1000 static signals allowing the user to examine multiple machine trains simultaneously.

The system is based on "Off-the-shelf" computer technology. This makes it uniquely easy for one or multiple plant operators to use, yet provides all the power and flexibility for in-depth engineering diagnostics.

The Zonic MMS is on the job 24 hours a day, 7 days a week, 52 weeks a year. If your critical machine train experiences an alarm or shutdown event at 2:00 AM, the system automatically retains all the machine information surrounding the event, ensuring that you have the complete story of what happened, no matter when the malfunction occurred.

For over 10 years Zonic has provided Turn-Key Machinery Monitoring Systems for the Petro-Chemical, Gas Transmission, and Electric Utility industries. The Zonic Machinery Monitoring System is an installed or transportable tool which provides important information about all of your critical machinery on a timely basis.

Benefits of Continuous Machinery Monitoring

  • Monitor critical machinery 100% of the time to recognize, alarm, and record critical events.
  • Reduce / minimize machinery commissioning time.
  • Reduce unplanned outages by early detection of developing problems.
  • Increase plant efficiency.
  • Reduce planned outage costs and facilities outage scheduling.
  • Provide accurate, flexible, and cost effective problem solving.
  • Flexible architecture supports various testing requirements.
  • Provides engineers with diagnostic information to improve safety and efficiency.

The Zonic MMS Offers Full Protection for Your Critical Machinery

Dynamic Data Acquisition of Critical Machinery

  • Simultaneous or multiplex acquisition from all displacement probes, velocity pickups, and accelerometers.
  • Software or hardware order tracking.
  • Simultaneous data acquisition and recording during machine transients.
  • User defined triggered data acquisition based on multiple events including time.

Process Static Data Acquisition

  • Continuous acquisition of process related data, (ie. pressure, temperature, and flow probes).
  • Interface to process control data highway.
  • User definable arithmetically determined variables.
  • User definable data acquisition rate.

Data Limit Checking

  • Amplitude Limit Checking
  • Trend Limit Checking
  • Rate of Growth Limit Checking
  • Multiple Limit Value
  • Spectral Limit Checking
  • User definable actions based on machine status

Data Analysis

  • Thermodynamic Performance Analysis
  • Statistical Reporting: Compressors, Gas Turbines, and pumps
  • Statistical data analyses - mean, range, standard deviation, frequency histogram and multi-linear regressions.
  • Interface to artificial intelligence packages.

Data Storage

  • Multi-Level Archives
  • Boxcar Archives
  • FIFO Files
  • Malfunction Files
  • Transient Files: start-up or shutdown event
  • All data files are user defined

Data Reporting

  • Real time or historical Data Displays - spectrum, time, waterfalls, orbit, bode, polar, strip chart, shaft center line, and scatter.
  • Multi-window displays.
  • Machine Train Graphics (Mimic Diagrams)
  • Production Summary Reports
  • Exception Summary Reports
  • Comparison Summary Reports
