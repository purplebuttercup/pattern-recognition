Direct Ship Service

The Direct Ship Program is now available to our Value Added Resellers. VARs can elect to have PCN configure the ports, pre-load current operating system and PCN Health Network application software, certify software and ship directly to any destination. Remember to submit a RISC or Intel Server Port Plan for each system you wish to have Direct Shipped and mail or fax it to PCN with your purchase order.
