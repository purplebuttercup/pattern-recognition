The files below are beta software for registered Layout v7.10 users
File Name
Date/Size
Description
IPC356.ZIP 9/10/97 - 18KB IPC-D-356 netlist option for Layout products. Creates industry-standard IPC-D-356 netlist directly from Layout. Several options exist for formatting the data as needed for different applications. This is beta software which will be released formally with the Layout v7.20 product release. There are no known bugs. A readme file in the ZIP file includes instructions for use.
IDF102.ZIP 11/11/97 - 40KB Updated IDF v2.00 interface for Layout Products. Fully bidirectional mechanical interface between OrCAD Layout and other programs that support this interface such as ProEngineer, SDRC. Supports mechanical ECO between mechanical software and OrCAD Layout. This is beta software which will be released formally with the Layout v7.20 product release. There are no known bugs. A readme file in the ZIP file includes instructions for use.
GCAD.ZIP 11/11/97 - 24KB GenCAD Interface for Layout Products. Outputs a Layout board database in an GenCAD v1.40 ASCII format suitable for use by fabrication and test software, such as Mitron's CB/Test, Fabmaster, and others. This is beta software which will be released formally with the Layout v7.20 product release. There are no known bugs. A readme file in the ZIP file includes instructions for use.

Before You Download

With any service pack or file update, we recommend that you make a backup copy of any files that might be replaced by those you download here. That way, should you need to restore the previous version you can.
