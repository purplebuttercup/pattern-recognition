Dataram Announces Memory Upgrades
For Compaq Professional Workstation 5100

Dataram is pleased to announce memory upgrades for Compaq Computer Corporation’s newest addition to it’s Windows NT-based workstation line.

The Professional Workstation 5100 features up to two 300MHz Pentium II processors and memory can be expanded to a maximum of 512MB. Like Compaq’s other workstations (6000 and 8000), the 5100 delivers increased memory bandwith thru the Highly Parallel System Architecture which utilizes dual memory controllers.

Dataram supports the 5100 workstation with upgrade capacities of 32MB, 64MB, 128MB and 256MB. Memory must be purchased and installed in pairs of identical capacity products.

Dataram's DRQ6080 memory upgrades are fully compatible with Compaq's memory and systems offering. Vendor part numbers are:
