PRODUCT INFORMATION

EPOCH 2000

The EPOCH 2000 command and control system lets you operate all of your satellites from a single workstation-based control center.

OASYS

OASYS is the high precision Orbit Analysis System - an integrated object oriented graphical environment for spacecraft mission analysis on today's high performance microcomputer work stations.

DRS

DRS is the DOMSAT Receive Station, for the collection, storage, and analysis of environmental data. The DRS is a PC-based system which includes an antenna, receiver, and processing software which allows customers to take advantage of the complete NOAA GOES Data Collection System.
