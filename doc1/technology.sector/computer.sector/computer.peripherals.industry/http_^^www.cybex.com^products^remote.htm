Recommended Usage: Computer configurations requiring remote access capability

PhoneBoot II allows you to boot your computer remotely from a touch-tone telephone or Hayes compatible modem. Once you hang up the line, your computer can issue or receive calls or perform shutdown routines before PhoneBoot II automatically turns the system off. A 4-digit access code provides protection against hackers and accidental booting, if desired.
 
Recommended Usage: Computer configurations requiring remote restart/recovery capability.

Use ReBoot II to automatically restart file servers, peripherals, and other electrical devices from a touch-tone telephone or Hayes-compatible modem. Chain multiple ReBoot II units together to restart up to 10 devices with a single phone call. A four-digit access code provides protection against hackers and accidental booting, if desired.
