Perle 833 Access Switch

Setting new standards in reliability and flexibility for central site remote access

  • Dial-in access for remote users
  • Powerful dial-out (modem pooling) for LAN based users
  • Supports up to 48 simultaneous users (T1/PRI)
  • Supports up to 60 simultaneous users (E1/PRI)
  • Redundant, hot swappable power supply
  • High performance multi-processor, multi-tasking architecture
  • Compact PCI® connector technology
  • Auto-sensing 10/100 Mbps Ethernet
  • Optional 4/16 bps token ring card
  • External loading design makes adding expansion cards fast and simple
